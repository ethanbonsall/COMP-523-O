# Ethan Repo

##  Tech Stack

- **Framework**: [Next.js](https://nextjs.org/)
- **UI**: React, Tailwind CSS
- **Backend**: Supabase (used for project data and storage)
- **Deployment**: Vercel
- **Language**: TypeScript
